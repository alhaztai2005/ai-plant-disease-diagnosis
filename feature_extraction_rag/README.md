# Feature Extraction Module (RAG)

This folder contains techniques used to convert text data into numerical representations.

## Files:
- `tf-idf.py`  
  Implements TF-IDF vectorization for text representation.

- `word_embedding.py`  
  Implements word embeddings (e.g., Word2Vec, GloVe, or MiniLM).

## Purpose:
- Convert text into vectors
- Enable similarity search in RAG
- Improve model understanding of semantic meaning
