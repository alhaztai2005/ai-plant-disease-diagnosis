# Model Module

This folder contains machine learning models used for classification and prediction.

## Files:
- `TFIDF-classification_model.py`  
  Classification model using TF-IDF features.

- `word_classifica_model.py`  
  Classification model using word embeddings.

- `train.ipynb`  
  Jupyter notebook for training and experimentation.

## Purpose:
- Train models on extracted features
- Perform predictions
- Evaluate model performance
