# RAG Application Module

This folder contains the implementation of the Retrieval-Augmented Generation (RAG) based application.

## Files:
- `FarmGenie QnA Dataset.csv`  
  Dataset containing question-answer pairs related to plant diseases.

- `app2.py`  
  Main application file (Flask/FastAPI) handling user queries and generating responses using RAG.

- `vector.py`  
  Handles vector database creation, embedding storage, and similarity search.

## Purpose:
- Retrieve relevant data from dataset
- Generate accurate responses using retrieved context
- Serve as backend for chatbot/web app
