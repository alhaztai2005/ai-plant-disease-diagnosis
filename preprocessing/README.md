# Preprocessing Module

This folder contains scripts responsible for cleaning and preparing raw data before it is used for feature extraction or model training.

## Files:
- `image_cleaning.py`  
  Handles image preprocessing such as resizing, normalization, noise removal, and formatting.

- `pos.py`  
  Performs Part-of-Speech (POS) tagging on textual data to extract grammatical structure.

## Purpose:
- Improve data quality
- Standardize inputs
- Prepare structured data for downstream tasks
